**Valentine's Interactive** is a web application designed to create a personalized, digital Valentine's Day experience.

## Features

### 1. Interactive Experience

The app greets the user with a beautiful landing page and guided journey:

- **The Quiz**: A fun relationship quiz (fully customizable) with celebratory confetti feedback.
- **Love Letter**: A deeply personal letter that animates line-by-line on parchment paper.
- **Virtual Card**: A 3D interactive greeting card that flips open to reveal a photo and message.

### 2. Creator Studio

Anyone can use the **Create** page to build their own Valentine link:

- Enter Sender & Recipient names.
- Write a custom love letter.
- Upload a personal photo for the card.
- Generate a unique link to share.

### 3. Sharing

- Generates a unique URL for the recipient.
- **WhatsApp Integration**: Built-in tool to instantly send the link via WhatsApp to your partner.

## How to Run

1.  **Install Dependencies**:

    ```bash
    npm install
    ```

2.  **Start Development Server**:

    ```bash
    npm run dev
    ```

3.  **Open in Browser**:
    Visit `http://localhost:5173`

## Tech Stack

- **Frontend**: React, Vite
- **Styling**: Vanilla CSS (Premium Glassmorphism Design), Tailwind support
- **Animations**: Framer Motion, Canvas Confetti
- **Icons**: Lucide React
- **Routing**: React Router DOM
